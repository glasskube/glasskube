# Installation

Glasskube is available for macOS, Windows and Linux.
Packages are available for [Homebrew](https://brew.sh/) and most package managers popular on Linux systems.

<Install/>

## Shell completion

Glasskube provides extensive autocomplete for many popular shells.
To take full advantage of this feature, please follow the steps for your shell below.

<Tabs groupId="shells">
  <TabItem value="Bash">
    To install completions in the current shell, run:
    ```
    source <(glasskube completion bash)
    ```
    For more information (including persistent completions), run:
    ```
    glasskube help completion bash
    ```
  </TabItem>
  <TabItem value="zsh" label="Z shell (zsh)">
    To install completions in the current shell, run:
    ```
    source <(glasskube completion zsh)
    ```
    For more information (including persistent completions), run:
    ```
    glasskube help completion zsh
    ```
  </TabItem>
  <TabItem value="fish" label="fish shell">
    To install completions in the current shell, run:
    ```
    glasskube completion fish | source
    ```
    For more information (including persistent completions), run:
    ```
    glasskube help completion fish
    ```
  </TabItem>
  <TabItem value="PowerShell">
    To install completions in the current shell, run:
    ```
    glasskube completion powershell | Out-String | Invoke-Expression
    ```
    For more information (including persistent completions), run:
    ```
    glasskube help completion powershell
    ```
  </TabItem>
</Tabs>
